Remplir la Base De Données

Ce document va expliquer les différentes étapes à effectuer pour remplir la base de données de la scène, coter utilisateur

Les tableurs d’entrées

Les données que l’ont souhaite avoir dans la scène sont à préciser dans les différents tableurs :

InputMeshs qui regroupe les différents objets (meshs), chaque objet a des paramètres à renseigner dont une référence à un séquenceur qui contient les sons !Les objets sont à regrouper par scène, et à espacer lors de la définition d’une autre scène

InputScene lie le 1er objet du groupe d’objet au nom de la scène, utile quand on prépare plusieurs scènes à l’avance.

InputSeq prépare les séquenceur, des listes de son(s) que l’on a attachés aux meshs précédemment aux sons qu’on va rentrer juste après !

InputSon permet de rentrer les sons dans la base de donnée, quelques paramètres sont à renseigner.

Attention les tableurs doivent être au format .csv pour le bon fonctionnement des scripts python

Exécuter le code python

pour remplir la base de données avec les informations des tableurs il suffit d’exécuter les 4 scripts python associés aux tableurs : InputMeshs py ,InputScene py , InputSeq py et InputSon py,
en utilisant la commande : python .\Input???.py
