Convertion BDD vers JSON

La génération de scène se faisant à partir d’un fichier au format JSON et le stockage de la scène et de ses éléments se faisant en SQL il faut réaliser un script réalisant l’interface.

Format du fichier JSON

Le format de la base de donnée est disponnible dans une documentation dédiée. Je vais donc ici détailler uniquement le format du fichier JSON.

Catégories

Le fichier est constitué de trois catégories d’éléments.

  • Meshs, il s’agit des objets qui seront créer dans la scène (arbres, sphère, etc …);

  • Séquenceurs, il s’agit des éléments chargés de gérer les sons, de les lancer un certain temps après une certaine action (ex: lancer un son d’oiseau 5 secondes après un clique sur un buisson);

  • sons, il s’agit des sons en eux même.

Composition d’un élément de type mesh

Un élément de type mesh est constitué de plusieurs informations:

  • son nom (qui est unique);

  • les noms des séquenceurs associés aux 4 actions possibles sur un mesh (clique, regard appuyé, entrée dans le nimbus, classique)

  • le rayon du nimbus associé à l’objet

  • les multipicateurs de tailles sur les 3 axes (un floatant)

  • le type de l’objet (important si aucun uri n’est fourni)

  • uri (chemin d’accès au fichier objet)

  • position en x,y,z

Composition d’un élément de type Séquenceur

  • un booléen servant à signaler si le séquenceur doit tourner en boucle

  • le nom (unique )

  • une liste de tuple à 3 éléments: la marge aléatoire autour du temps de déclenchement , le nom du son, le temps de déclenchement

Composition d’un élément de type son

  • nom (unique)
  • un booléen servant à signaler si le son est spatialisé
  • uri ( nom du fichier)
  • volume (entre 0 et 1)

Interface BDD vers JSON

Le programme prend en entrée un nom de scène.

Il vas chercher en bdd le nom du premier mesh associé a cette scène.

Grâce à ce nom, il charge un a un tout les meshs de la scène.

Il charge ensuite chaque séquenceur associé à un de ces meshs.

Enfin il charge tout les sons de ces séquenceurs

Utilisation

Pour utiliser le programme d’interface exécuter, depuis le dossier contenant le programme, la commande : python3 bdd_to_json.py {nom de la scène}
